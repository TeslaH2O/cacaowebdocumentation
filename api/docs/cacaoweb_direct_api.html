cacaoweb for videos or files publishers

Uploading files

To distribute your files through the cacaoweb network, you have two options :

  • Upload your files through cacaoweb interface

    • In this case, cacaoweb will upload your file as it is on the cacaoweb network. If you're uploading videos for streaming, your video needs to be in a format compatible with Flash streaming, which is :

    • FLV ; or
    • MP4 with metadata at the beginning of the file. (if you don't know how to do it, it will be easier to use cacaoweb second upload option futher down)

    Once you have your files ready, and provided cacaoweb is running on your computer, you can upload them through the cacaoweb web interface on the administration page. Or you can also use the command line interface with the -u or -upload switch : 

    	$ ./cacaoweb -u MyFile1.zip -u MyNewMovie-Bdrip.mp4
	$ ./cacaoweb -upload myTVShow.S05E14.mp4

    Or you can use cacaoweb's web API like this : 

    	 $ wget http://localhost:4001/upload?filepath=/home/JohnDoe/Movies/MyNewMovie-Bdrip.mp4
	 $ wget http://localhost:4001/upload?filepath=/home/JohnDoe/MyFiles/myTVShow.S05E14.mp4

    At any time you can check your uploads status by querying the page : 

    	 $ wget http://localhost:4001/uploads

    Which will provide you with an XML document featuring all your current uploads and various information regarding their status.

    Once the file has been successfully uploaded, you can get its cacaoweb link by going to the cacaoweb administration interface and click on "show link", or you can figure it out by yourself : 

    	 $ ./cacaoweb -md5 MyFile1.zip
	 $ ./cacaoweb -md5 MyNewMovie-Bdrip.mp4

    and then the link will just be http://localhost:4001/?f=MD5_OF_THE_FILE where MD5_OF_THE_FILE should be replaced with the actual MD5 of the file.

    After having successfully uploaded a file, you might want to remove the upload. To do this :

    	 $ wget http://localhost:4001/removeupload?f=MD5_OF_THE_FILE

    and the upload will be removed (but the file is still accessible in the cacaoweb network).

  • Use a cacaoweb uploading service like our partner H264Stage.com that will do everything for you automatically.

    A hosting service will take care for you of the file format transcoding to make any video available for streaming and will upload the video automatically, providing you at the end with the cacaoweb link.

There is no way to remove a file from the cacaoweb network.

Sharing the link

Once you have uploaded a file or a video and you got its link, either through the option "Show link" if you uploaded the file through cacaoweb, or in the h264stage.com interface if you used h264stage.com to upload your file, you can share the link with all your friends, family, or even put the link on links directories on the internet.

There are two types of links provided by cacaoweb:

  • a direct link, like this one http://localhost:4001/?f=20B5A05913E0825BF423BE968E31F527. People can click on this link and the file will be download automatically.
  • an embed link. It's a whole html code used for videos or music files so the content can be played in Flash on the host website. Typically, you would use the embed link when you want to add your link to a link directory website, so they can integrate it in their website. Internet links directories usually ask you for this type of link so they can integrate it in their website.
